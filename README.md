# -Cafe-Management-System.
A desktop-based Cafe Management System developed using Python  and MySQL to automate cafe operations such as customer management, billing, sales tracking, payment processing, and inventory management.
Project Overview

This project demonstrates the integration of Python GUI, MySQL Database, and SQL queries to build a functional cafe billing application. The system generates invoices, updates customer balances, records transactions, and stores sales information in a relational database.

Features
Customer Management
Cafe Billing & Invoice Generation
MySQL Database Integration
Sales & Transaction Recording
Account Balance Management
Inventory Management (Vegetables & Spices)
Salary Management for Employees
Payment Methods (Cash, UPI, Card)
User-friendly GUI built with Tkinter
